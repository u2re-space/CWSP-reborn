# Platform Notes Compatibility Directory

`platforns` is a historical misspelling and `docs/drivers` currently resolves to
this directory. It remains temporarily to avoid breaking existing links.

Canonical cross-platform capability documentation is
[`../Drivers.md`](../Drivers.md). Platform-specific implementation notes should
not be added here until the symlink-normalization task selects and migrates a
correctly named `docs/platforms/` canonical directory.

This directory contains no driver implementation and does not imply platform readiness.
